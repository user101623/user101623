## 🌟 Featured Project

### 🚀 EzResume
> **A full-stack web application that uses artificial intelligence to transform raw form data into an ATS-friendly resume in seconds.**

| Frontend | Backend | Deployment / Hosting |
| :--- | :--- | :--- |
| Javascript (React), Vite, Tailwind CSS | Python 3.14 | Firebase, Vercel |

#### Key Features:
* ⚡ **Feature 1:** Orchestrated an AI pipeline utilizing multiple Gemini models (including Gemini 3.5 Flash, Gemini 3 Flash Preview, and Gemini 2.5 Flash-Lite) to parse raw data and generate ATS-friendly resumes in seconds.
* 🧠 **Feature 2:** Engineered a seamless, highly intuitive user interface focused on optimizing the user journey from form input to final resume download.
* 🗄️ **Feature 3:** Integrated Firebase Authentication and Firebase Realtime Database to securely manage user login credentials and synchronize user profile data.

👉 [Check out the Live Demo!](https://www.ezresumeai.vercel.app)
